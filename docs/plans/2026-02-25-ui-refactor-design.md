# UI重构设计方案

**日期**: 2026-02-25
**主题**: UI风格重构、移除习惯Tab、调整表单顺序

## 需求背景

1. **UI风格调整** - 对标现代深色主题任务App
2. **移除习惯Tab** - 合并到事务中
3. **表单顺序调整** - 事务类型移到最上方

---

## UI风格设计

### 整体风格
- 深紫色/炭黑色背景 (#1a1a2e)
- 柔和粉彩色调accent（淡紫色、珊瑚色、青色）
- 大圆角卡片 (16-20px)
- 线条图标
- 简洁现代设计

### 颜色系统

| 用途 | 颜色 |
|------|------|
| 主背景 | #1a1a2e (深紫色) |
| 卡片背景 | #252540 (稍浅紫) |
| 强调色1 | #a78bfa (淡紫色) |
| 强调色2 | #fb7185 (珊瑚色) |
| 强调色3 | #22d3d3 (青绿色) |
| 文字 | #ffffff / #a0a0a0 |
