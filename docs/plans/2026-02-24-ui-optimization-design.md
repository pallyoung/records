# UI 优化设计文档

## 概述

在保持"日记手账风"温暖质感的基础上，全面提升 UI 精致度和交互体验。

---

## 一、设计原则

- 保持温暖色调，增加视觉层次
- 圆角适中（12-16px），更加干练
- 维持现有内边距
- 添加流畅的过渡动画

---

## 二、色彩系统优化

### 2.1 主色调

保持现有的温暖米色主调：
- `--bg-primary: #FAF8F5`
- `--bg-secondary: #F5F2ED`
- `--bg-card: #FFFFFF`

### 2.2 阴影层次

增强阴影效果：
```scss
--shadow-sm: 0 2px 8px rgba(61, 51, 41, 0.06);
--shadow-md: 0 4px 16px rgba(61, 51, 41, 0.10);  // 加深
--shadow-lg: 0 8px 32px rgba(61, 51, 41, 0.15);  // 加深
```

---

## 三、组件优化

### 3.1 卡片组件

- 圆角：12px
- 添加细微边框：`border: 1px solid rgba(61, 51, 41, 0.08)`
- 悬浮时：`transform: translateY(-4px)` + 加强阴影

### 3.2 按钮

- 圆角：8px
- 悬浮：`transform: scale(1.02)` + 颜色变深
- 点击：`transform: scale(0.98)`

### 3.3 输入框

- 圆角：8px
- 聚焦时边框颜色加深
- 添加过渡动画

---

## 四、动画系统

### 4.1 过渡时长

```scss
--transition-fast: 150ms ease;
--transition-normal: 250ms ease;
--transition-slow: 350ms ease;
```

### 4.2 动画效果

1. **页面切换**：淡入淡出 250ms
2. **卡片悬浮**：上浮 + 阴影增强 200ms
3. **按钮交互**：缩放 150ms
4. **列表出现**：依次淡入（stagger 50ms）
5. **骨架屏**： shimmer 动画

---

## 五、视觉层次

### 5.1 间距系统

保持现有间距，添加分组：
- `--space-xs: 4px`
- `--space-sm: 8px`
- `--space-md: 16px`
- `--space-lg: 24px`
- `--space-xl: 32px`

### 5.2 文字层次

- 标题：`--text-primary` 加粗
- 正文：`--text-primary`
- 次要信息：`--text-secondary`
- 辅助信息：`--text-muted`

---

## 六、移动端优化

- 保持现有内边距
- 触摸区域 ≥ 44px
- 优化底部导航空间
- 手势支持（滑动删除等）

---

## 七、实施计划

1. **基础样式更新** - 更新 index.css 变量
2. **组件动画** - 添加过渡效果
3. **列表动画** - 添加入场动画
4. **交互优化** - 按钮/卡片交互效果
5. **细节打磨** - 边框、阴影等
